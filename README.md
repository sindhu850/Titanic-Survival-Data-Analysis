# Titanic-Survival-Data-Analysis
Exploratory Data Analysis of Titanic dataset using python to identify survival patterns based on gender, age, class, and other Factors.
Tools: Python, Pandas, NumPy, Matplotlib, Seaborn*Project Overview:  
Conducted an in-depth Exploratory Data Analysis (EDA) on the Titanic dataset to explore passenger demographics, travel patterns, and factors influencing survival rates during the sinking of the RMS Titanic.
Key Contributions:-Data Cleaning: Handled missing values in critical columns (Age, Cabin, Embarked) to prepare the dataset for analysis.
Demographic Analysis: Analyzed passenger demographics, including age distribution, gender breakdown, and class representation.- *Deck Location Investigation: Examined cabin data to assess the impact of deck location on passenger survival rates.
Embarkation Analysis: Investigated embarkation points to understand passenger boarding patterns.
Family Relationship Study: Analyzed family connections (SibSp and Parch) to identify those traveling alone versus with family, and studied their correlation with survival rates.
Visualization: Created insightful visualizations using Seaborn and Matplotlib, including count plots, catplots, and distribution charts, to illustrate findings.
Insights: Identified significant patterns indicating that higher passenger class and upper deck locations positively influenced survival rates, while traveling with family slightly increased survival chances.
Key Insight:  
Passengers in higher classes and upper decks had better survival chances, while those traveling with small families had slightly higher survival rates compared to solo travelers.
